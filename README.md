# 이예다 (LEE YEDA) 팬페이지

SOOP 버추얼 스트리머 **이예다** 공식 팬페이지.

- 페이지: 메인 · 프로필 · 옷장 · 업보 · 시참권 · 스탬프 (+ `admin/` 관리자)
- 스택: 정적 HTML/CSS/JS + Supabase + Cloudflare Pages
- Supabase 키·관리자 비밀번호 입력 완료 (비밀번호는 admin/index.html에서 변경 가능)
- ⚠️ **v2 업데이트 후 최초 1회**: Supabase SQL Editor에서 `supabase_patch_v2.sql` 을 한 번 Run (업보 사용기한·시참권·스탬프 테이블)
